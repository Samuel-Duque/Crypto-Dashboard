# 🚀 Crypto Dashboard

Un dashboard de criptomonedas desarrollado con React, TypeScript y la API de CoinGecko.

El objetivo de este proyecto es practicar el consumo de APIs, el manejo de estados, la tipificación con TypeScript y la construcción de interfaces modernas para visualizar información financiera en tiempo real.

---

## 📖 Descripción

Crypto Dashboard permite consultar información actualizada sobre criptomonedas utilizando la API de CoinGecko.

Actualmente el proyecto se encuentra en desarrollo y se irá ampliando progresivamente con nuevas funcionalidades.

## 📦 Instalación

Clona el repositorio:

```bash
git clone https://github.com/Samuel-Duque/Crypto-Dashboard.git
```

Ingresa al directorio:

```bash
cd Crypto-Dashboard
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el proyecto:

```bash
npm run dev
```

---

## 🔑 Variables de entorno

Crea un archivo `.env` en la raíz del proyecto:

```env
VITE_API_URL=https://api.coingecko.com/api/v3
VITE_API_KEY=TU_API_KEY
```
